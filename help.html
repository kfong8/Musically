Notes - Written as dots with/without tails on a 5 line staff (look defines length).
b - Used to designate a note a half-step below the one written (often black key).
# - Used to designate a note a half-step above the one written (often black key).

Octaves - 7 sets of {C,D,E,F,G,A,B} that full set of notes is divided into.
8va - Used to designate a note one octave above the one written.
8vb - Used to designate a note one octave below the one written.

Treble Clef - Signature designating staff holds notes from octaves 4-7.
Bass Clef - Signature designating staff holds notes from octaves 1-3.
